---
description: Guarda y sube todos los cambios a GitHub al finalizar una sesión de trabajo.
---
Este workflow se encarga de guardar todo tu trabajo local y subirlo a GitHub.

// turbo-all
1. Ejecuta el comando de terminal: `git add .`
2. Ejecuta el comando de terminal: `git commit -m "Auto-sync: Guardando progreso de la sesión"`
3. Ejecuta el comando de terminal: `git push`
